# Introduction

Minecraft UI is a library of UI components built with React that implements an **unofficial Minecraft Design System**.

## Instalation

Minecraft UI is available as an [npm package](https://www.npmjs.com/package/minecraft-react-ui).

To install and save in your package.json dependencies, run the command below using npm:

```
npm install minecraft-react-ui
```

Please note that [react](https://www.npmjs.com/package/react) >= 18.0.0 and [react-dom](https://www.npmjs.com/package/react-dom) >= 18.0.0 are peer dependencies.

## Minecraft font

Minecraft UI was designed with the Minecraft font in mind. So be sure to follow these instructions to use Minecraft UI:

### Standard

```
<link href="https://fonts.cdnfonts.com/css/minercraftory" rel="stylesheet" />
```

### @import

```
<style>@import url('https://fonts.cdnfonts.com/css/minercraftory');</style>
```

### @font-face

```
<style>
@font-face {
    font-family: 'Minercraftory';
    font-style: normal;
    font-weight: 400;
    src: local('Minercraftory'), url('https://fonts.cdnfonts.com/s/46126/Minercraftory.woff') format('woff');
}
</style>
```

### Specify in CSS

```
font-family: 'Minercraftory', sans-serif;
```
